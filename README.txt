This plugin extract a compressed (zipped) sparx file to upload folder.

== Description ==
With this plugin you can upload a compressed sparx file (zip). The plugin will extract the files to a sparx folder and
the sparx project is accessible for a user by entering the sparx projects url.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sk-sparx-upload.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
